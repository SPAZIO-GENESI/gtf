# processes/ — Processi (PRC)

Un file `PRC-<slug>.yaml` per ogni processo ricorrente, con owner, trigger,
output ed evidenza prodotta, incluso il costo umano dichiarato
(`human_cost_minutes_month`, principio PRN-10). Validato da
`schemas/process.schema.json`.

Popolato in fase **M1 — Import dell'esistente** (vedi ARCHITECTURE.md §12).
