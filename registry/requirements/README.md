# requirements/ — Requisiti normativi adottati (REQ)

Un file `REQ-<slug>.yaml` per ogni clausola normativa pertinente, validato da
`schemas/requirement.schema.json`. Vedi GTF-ARCH §5 (Compliance Framework) e
§3.3 per un esempio completo.

Popolato in fase **M2 — Compliance Map** (vedi ARCHITECTURE.md §12).
