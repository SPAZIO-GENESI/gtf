# actions/ — Azioni correttive/migliorative (ACT)

Un file `ACT-<anno>-<progressivo>.yaml` per ogni azione con scadenza, nata
da un incidente, un rischio o una decisione. Validato da
`schemas/action.schema.json`. Le ACT scadute e ancora aperte compaiono come
banner nel Trust Center (PRN-09).

Popolato in fase **M4 — Ciclo vivo**, o prima se serve tracciare un'azione
concreta (vedi ARCHITECTURE.md §12).
